Multi-agent systems, tool use, RAG, orchestration of Claude / GPT / Gemini / DeepSeek. From PoC to production. Comfortable with stateful conversations, long context, structured output, and cost optimization.

### Voice assistants
STT + TTS + LLM pipelines. Whisper, Web Speech API, edge-tts. PWAs, Chrome extensions, 3D avatars (VRM). Real-time interaction with mood / animation feedback.

### AI-driven automation
Putting LLMs into business workflows: email triage, document processing, candidate screening, report generation, Telegram chatbots. Cost-aware design.

### AI-driven testing
End-to-end + Visual Regression with LLM analysis of failures. Useful for large web apps where flake-vs-bug discrimination is expensive.

### Algo trading and analytics
Strategy development, backtesting, risk management. Bybit / Binance APIs, ClickHouse, Python.

### AI consulting
Embedding LLMs into existing products. Where they help, where they don't, how to measure ROI, how to keep cost predictable.
